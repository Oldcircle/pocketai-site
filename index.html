What is PocketAI?

PocketAI puts the command-line coding agent running on your own Mac right in your pocket. It renders a real, native conversation on your phone — message bubbles, thinking blocks, tool calls, Markdown, and code — and lets you keep working from anywhere, even on cellular.

  • From anywhere. Your Mac opens a secure encrypted tunnel, so your phone reaches it over cellular or any Wi-Fi — no VPN, no port forwarding.
  • You stay in control. Every sensitive action asks you first: Allow, Always allow, or Deny — with diff previews for file edits.
  • Private by design. No account, no analytics, no data collection. There is no PocketAI server.

PocketAI is a client for a coding agent that runs on your own Mac. It works with Claude Code and similar CLI coding agents. PocketAI is an independent app and is not affiliated with any third-party agent or model provider.
